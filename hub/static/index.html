API Modules

Three active modules

Phase 1 modules are live. Full API reference with request/response schemas is available in the interactive documentation.

POST /api/v1/auth/

Steward registration, login, and session management. JWT-based authentication for all protected endpoints. Role-based access control: steward, contributor, admin.

GET /api/v1/activities/

Activity stream — records of steward actions across the commons. Feeds the Protocol layer's governance and the UBECrc reciprocity reward system.

POST /api/v1/observations/

Phenomenological observations from Living Labs. Stores qualitative and sensor data together — Goethean Science in structured form. Seeded with the 13 Questions to the Soil.

GET /api/v1/maps/  —  Phase 2

Bioregional GIS data pipeline connecting sensor observations to the UBEC Maps service. Watershed intelligence layer. Planned for Phase 2.

Token Economy

Hub as reward engine

Every verified observation submitted through the Hub triggers UBECrc reciprocity token rewards on the Stellar network. The Hub is where data quality becomes economic value.
