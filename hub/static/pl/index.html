Moduły API

Trzy aktywne moduły

Moduły Fazy 1 są aktywne. Pełna dokumentacja API ze schematami żądań/odpowiedzi dostępna jest w interaktywnej dokumentacji.

POST /api/v1/auth/

Rejestracja opiekuna, logowanie i zarządzanie sesjami. Uwierzytelnianie oparte na JWT dla wszystkich chronionych punktów końcowych. Kontrola dostępu oparta na rolach: opiekun, współtwórca, admin.

GET /api/v1/activities/

Strumień aktywności — zapisy działań opiekunów w commons. Zasila zarządzanie warstwy protokołu i system nagród wzajemności UBECrc.

POST /api/v1/observations/

Obserwacje fenomenologiczne z Living Labs. Przechowuje dane jakościowe i sensorowe razem — Nauka Goethańska w ustrukturyzowanej formie. Uzupełniona o 13 pytań do gleby.

GET /api/v1/maps/  —  Phase 2

Bioregional GIS data pipeline connecting sensor observations to the UBEC Maps service. Watershed intelligence layer. Planned for Phase 2.

Ekonomia tokenów

Hub jako silnik nagród
