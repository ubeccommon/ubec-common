# UBEC Commons — Platform Monorepo

> *"I am because we are. Place. Protocol. Planet."*

Web 4.0 bioregional DAO protocol network rooted in Ubuntu philosophy.

## Structure

```
ubec-commons/
├── portal/           ubec.network             — static HTML gateway
├── protocol/         bioregional.ubec.network — DAO token engine
├── maps/             mapservice.ubec.network  — Mapbender GIS
├── living-labs/      living-labs.ubec.network — citizen science
├── hub/              iot.ubec.network         — central API
├── erdpuls/          erdpuls.ubec.network     — flagship Living Lab
├── open/             ubeccommon.github.io     — developer commons
├── learn/            .../Pattern_Language_of_Place/
└── design-cdn/       design.ubec.network      — shared design system
```

## Quick start

Each service is self-contained. See `<service>/README.md` for setup.
